Skill 1 — Secure Guard

Secure Guard — pare-feu IA

sécurité RGPD
📍 Actif dans toutes vos conversations — détecte les données sensibles avant envoi

Ce skill surveille silencieusement ce que vous envoyez à Claude. Si vous partagez un document contenant des données sensibles (noms de clients, IBAN, données RH…), il vous alerte avant d'aller plus loin et propose une version anonymisée.

Prompt à coller dans le Skill Creator — Settings → Features → Create a skill
Crée un skill de sécurité RGPD pour une dirigeante de TPE française. Ce skill s'appelle "Secure Guard". Il surveille silencieusement tout contenu que je partage dans Claude. Comportement automatique dès qu'un document, texte ou email est partagé : Scanne le contenu et identifie les données sensibles RGPD : - Noms et coordonnées de personnes identifiables - Données financières (IBAN, montants, contrats) - Données RH (salaires, évaluations, numéros de sécurité sociale) - Informations stratégiques (brevets, devis en cours, prix fournisseurs) - Données de santé Si aucune donnée sensible : traite normalement, sans interruption. Si données sensibles détectées, affiche avant tout traitement : 🟡 SECURE GUARD — Données sensibles détectées Liste exactement ce qui est sensible et pourquoi. Propose une version avec les données remplacées : [PRÉNOM], [MONTANT], [NOM-ENTREPRISE]… Demande confirmation avant de continuer. Si le contenu est clairement dangereux (données de santé, IBAN avec contexte financier) : 🔴 SECURE GUARD — Ne pas traiter Explique pourquoi et propose une alternative sécurisée. Ne jamais demander de cliquer sur un lien pour vérifier. Mode silencieux par défaut — n'intervient que si risque détecté.
Skill 2 — Amélioration continue

Amélioration continue

bilan hebdo / mensuel
📁 À lancer en fin de semaine — analyse votre façon de travailler avec l'IA

Ce skill analyse vos conversations récentes et vous dit concrètement comment vous progressez dans votre usage de l'IA — ce qui fonctionne, ce qui freine, ce que vous pourriez automatiser. Deux rythmes : hebdomadaire (5 min) et mensuel (20 min).

Prompt à coller dans le Skill Creator — Settings → Features → Create a skill
Crée un skill d'amélioration continue IA pour une dirigeante de TPE française. Ce skill s'appelle "Amélioration continue". Il analyse mes conversations récentes et m'aide à progresser dans mon usage de l'IA. Quand je demande mon bilan hebdomadaire : Analyse mes conversations de la semaine. Identifie comment j'utilise l'IA : tâches déléguées, façon de formuler mes demandes, résultats obtenus. Réponds en 3 points — 200 mots max : ✅ CE QUI FONCTIONNE BIEN 2 usages concrets où j'exploite bien l'IA. Cite un exemple réel de nos échanges. 🔧 CE QUI POURRAIT ÊTRE AMÉLIORÉ 2 points précis : prompts trop vagues, tâches manuelles que l'IA pourrait faire, occasions manquées. 🎯 MON ACTION DE LA SEMAINE Une seule chose à changer dès maintenant. Formulée en action, pas en conseil. Termine par : "Tu veux qu'on approfondisse l'un de ces points ?" Quand je demande mon bilan mensuel : Bilan de fond en 4 points : 1. Mes outils IA sont-ils toujours conformes RGPD + AI Act ? 2. L'équipe applique-t-elle le réflexe DDO (Département → Donnée → Outil) ? 3. Y a-t-il une échéance légale dans les 30 jours ? 4. Mes 3 actions prioritaires pour le mois prochain. Produis un compte-rendu daté que je peux conserver.
Skill 3 — Veille réglementaire (optionnel)

Veille IA — hebdo & mensuelle

AI Act · RGPD · outils
📰 Reste informée des évolutions IA sans y passer du temps

Ce skill vous donne en 200 mots l'essentiel de l'actualité IA qui impacte votre entreprise — ce qui a changé, ce à quoi il faut faire attention, une action concrète. Hebdo ou mensuel selon votre rythme.

Prompt à coller dans le Skill Creator — Settings → Features → Create a skill
Crée un skill de veille IA pour une dirigeante de TPE française. Ce skill s'appelle "Veille IA". Il me tient informée des actualités IA et réglementaires qui impactent mon entreprise. Quand je demande ma veille hebdomadaire — 200 mots max : 📰 L'ACTU DE LA SEMAINE Une info IA ou réglementaire qui impacte les TPE françaises cette semaine. Si rien de notable : confirme en une phrase. ⚠️ SIGNAL À SURVEILLER Un nouvel outil, une faille de sécurité, ou un changement réglementaire à garder à l'œil. ✅ MON ACTION DE LA SEMAINE Une seule chose concrète à faire dans ma gouvernance IA cette semaine. Quand je demande ma veille mensuelle : 🔍 Mes outils IA sont-ils toujours conformes RGPD + AI Act ? 📅 Y a-t-il une échéance légale dans les 30 jours ? 👥 Ma politique IA est-elle à jour ? 🎯 3 actions pour le mois prochain, par ordre de priorité. Produis un compte-rendu daté que je peux conserver.
Créez vos propres skills — le prompt qui propose

Laissez Claude concevoir vos skills sur mesure

Ouvrez le Skill Creator, collez ce prompt. Claude vous pose 5 questions, analyse votre façon de travailler, et génère les skills adaptés à votre profil — prêts à installer.

Prompt à coller dans le Skill Creator — pour créer vos skills personnalisés
Je suis dirigeante d'une TPE française. Je veux créer des skills IA personnalisés adaptés à mon travail. Commence par me poser 5 questions courtes pour comprendre : - Mon secteur et la taille de mon équipe - Comment j'utilise Claude au quotidien (fréquence, types de tâches) - Quelles tâches répétitives je fais souvent et que je pourrais déléguer - Ce qui me frustre dans mes interactions actuelles avec l'IA - Les résultats que j'aimerais obtenir automatiquement sans y penser Ensuite, propose-moi 3 skills personnalisés. Pour chacun : 1. Un nom clair et ce qu'il fait concrètement 2. Le comportement exact que tu vas lui donner 3. Génère-le directement — ne demande pas de validation Commence par les questions. Sois direct.
